Overview

The aoi.js RAG API validates, analyzes, and explains aoi.js code using a Retrieval-Augmented Generation pipeline built directly on official documentation.

Endpoints

Method Path Description Example
GET /api Lists all available endpoints, descriptions, and example usage. /api
GET /api/basicQuery Answers aoi.js questions strictly from local docs using RAG. /api/basicQuery?request=What+is+aoi.js
GET /api/function Returns syntax, description, parameters, examples for a given aoi.js function. /api/function?name=addButton
GET /api/generateStrict Strict aoi.js code generator using only local docs. Returns a single code block. /api/generateStrict?request=Economy+daily+command
GET /api/validateAoi Validates aoi.js code (syntax, docs, logic). Optional minimal fix with mode=fix. /api/validateAoi?code=```$sendMessage[hello;false]```&mode=fix

Requires a valid API key generated via the Dashboard.
